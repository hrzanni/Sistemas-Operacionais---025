SLEEP()

A primitiva sleep() é uma chamada de sistema usada em sistemas operacionais para fazer o processo atual pausar a sua execução por um determinado período de tempo. Durante esse tempo, o processo não realiza nenhuma ação e não consome ciclos de CPU. A função sleep() é útil quando você deseja que um processo aguarde antes de continuar a execução, sem precisar de intervenção ativa.

Ilustração da chamada de sistema sleep()



Veja o exemplo de código a seguir da função sleep():


  #include <stdio.h>  
  #include <sys/time.h>    
  #include <unistd.h>    
  
  int main() {
    for(int i = 1; i < 6; i++) {
	    printf("\r%d", i);
	    fflush(stdout);
	    if (i < 5) {
        sleep(1);
	    }
    }  
    
    printf("\n");
    return 0;
}
