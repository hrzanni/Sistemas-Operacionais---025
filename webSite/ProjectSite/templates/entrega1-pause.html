PAUSE()

A primitiva pause() é uma chamada de sistema em sistemas operacionais baseados em UNIX e Linux que faz com que o processo atual entre em um estado de espera até que receba um sinal (signal) de outro processo ou do próprio sistema. É uma função simples, mas muito útil para sincronização e controle de fluxo em programas que precisam aguardar a chegada de eventos assíncronos, como sinais de interrupção.

Ilustração da chamada de sistema pause()



Veja o exemplo de código a seguir da função pause():


  #include <stdio.h>  
  #include <signal.h>    
  #include <unistd.h>  
  
  void handle_signal(int sig) {
    printf("\nSignal %d received\n", sig);
    fflush(stdout);  
  }

  int main() {
    setbuf(stdout, NULL);  

    printf("Press (Ctrl+Z) to send a signal\n");

    signal(SIGTSTP, handle_signal);

    pause();

    return 0;
  }
